# AnimeVerse AI ⚔️

AnimeVerse AI is an AI-powered anime companion built with Streamlit.

## Features

- 🔍 Anime Search with Posters & Details
- 🎌 AI Anime Recommender
- ⚔️ Anime Battle Simulator
- ✨ Anime Quote Generator
- 🎭 Anime Personality Quiz

## Tech Stack

- Python
- Streamlit
- Jikan Anime API
- Gemini AI

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
